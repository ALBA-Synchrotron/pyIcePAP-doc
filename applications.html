PyIcepPAP console application¶

usage: pyIcePAP [-h] {save,check,update,autofix,send} ...

Sub-commands:¶

save¶

Command to save the configuration to a file

pyIcePAP save [-h] [-p PORT] [-t TIMEOUT] [--bkpfile BKPFILE] [-d]
              host [axes [axes ...]]

Positional Arguments¶

host IcePAP Host
axes

Axes to save, default all

Default: []

Named Arguments¶

-p, --port

IcePAP port

Default: 5000

-t, --timeout

Socket timeout

Default: 3

--bkpfile

Output backup filename

Default: “”

-d, --debug

Activate log level DEBUG

Default: False

check¶

Command to check the IcePAP configuration for a backup file

pyIcePAP check [-h] [-d] [--host HOST] filename [axes [axes ...]]

Positional Arguments¶

filename Backup file
axes

Axes to save, default all

Default: []

Named Arguments¶

-d, --debug

Activate log level DEBUG

Default: False

--host

Use another IcePAP host instead of the backup saved.

Default: “”

update¶

Command to change the firmware version. It creates a backup before to change the FW

pyIcePAP update [-h] [-p PORT] [-t TIMEOUT] [--bkpfile BKPFILE]
                [--no-check NOCHECK] [-d]
                host fwfile

Positional Arguments¶

host IcePAP Host
fwfile Firmware binary file

Named Arguments¶

-p, --port

IcePAP port

Default: 5000

-t, --timeout

Socket timeout

Default: 3

--bkpfile

Output backup filename

Default: “”

--no-check Avoid the checking procedure after the update
-d, --debug

Activate log level DEBUG

Default: False

autofix¶

Command to autofix a driver configuration

pyIcePAP autofix [-h] [-p PORT] [-t TIMEOUT] [--bkpfile BKPFILE]
                 [--no-check NOCHECK] [-d]
                 host loadfile

Positional Arguments¶

host IcePAP Host
loadfile backup to load

Named Arguments¶

-p, --port

IcePAP port

Default: 5000

-t, --timeout

Socket timeout

Default: 3

--bkpfile

Output backup filename

Default: “”

--no-check Avoid the checking procedure after the update
-d, --debug

Activate log level DEBUG

Default: False

send¶

Command to send IcePAP raw commands

pyIcePAP send [-h] [-p PORT] [-t TIMEOUT] [-d] host command

Positional Arguments¶

host IcePAP Host
command Raw command

Named Arguments¶

-p, --port

IcePAP port

Default: 5000

-t, --timeout

Socket timeout

Default: 3

-d, --debug

Activate log level DEBUG

Default: False

Communication layer¶

class pyIcePAP.communication.EthIcePAPCommunication(host, port=5000, timeout=3)¶

Class implementating the socket communication layer for the Base Icepap motor controller class.

class pyIcePAP.communication.IcePAPCommunication(comm_type, *args, **kwargs)¶

This abstract class provides an abstraction of the communication layer for the IcePAP motion controller by defining a common API.

get_comm_type()¶

Returns the communication type implemented in the controller.

Returns:CommType object according to the communicationlayer defined.
send_binary(ushort_data)¶

Method to send a binary data to the IcePAP controller.

Parameters:ushort_data – Data converted to a unsigned short list.
send_cmd(cmd)¶

Method to send commands to the IcePAP controller. It uses acknowledge communication (IcePAP User Manual pag. 37).

Parameters:cmd – Command without acknowledge character and CR and/or LF.
Returns:None or list of string without the command and the CRLF.
